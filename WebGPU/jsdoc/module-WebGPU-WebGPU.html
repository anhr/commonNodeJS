(static) out2Matrix(out, settingsopt)

Converts the out array to matrix.
Parameters:
Name Type Attributes Default Description
out ArrayBuffer out ArrayBuffer. See settings.out param of WebGPU class for details.
settings object <optional>
 {} The following settings are available
Properties
Name Type Attributes Default Description
type object <optional>
 Float32Array type of the out ArrayBuffer. Allowed Float32Array and Uint32Array.
size Array <optional>
 size of result matrix. 
size.length is dimension of result matrix.
size[0] is first dimension.
---
size[i] is next dimension.
---
size[size.length - 1] is last dimension.
Esample:
[
  10,//rows count
  2//columns count
]

creates two dimesional matrix with 10 rows and 2 columns.

If size is undefined, then dimension and size of result matrix must be defined in the header of the out:
First item of the out is dimension of result matrix.
Second item of the out is first dimension.
---
Next item of the out is next dimension.
---
dimension item of the out is last dimension.
Example:

const array = new Float32Array(out);

if

array[0] = 2//two dimesional matrix
array[1] = 10//rows count
array[2] = 2//columns count

then result matrix is two dimensional matrix with ten rows and two columns.
push function <optional>
 function(item). item - new matrix item. Called when a new matrix item is ready. You can add a new item to your matrix. The result matrix is empty if you have added push to the setting and settings.returnMatrix is not true.
returnMatrix boolean <optional>
 true - result matrix is not empty. Has effect only if settings.push is defined.
