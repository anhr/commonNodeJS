async(url, optionsopt) → {string}

Load file asynchronously
Parameters:
Name Type Attributes Description
url string URL of an external file.
options Object <optional>
 the following options are available.
Properties
Name Type Attributes Description
onload function <optional>
 function () The onload event occurs when a script has been loaded.
onerror onerror <optional>
 function ( str ) The onerror event occurs when an error has been occured.
Source:
  • index.js, line 314
Returns:
file content
Type
string
Examples
//Simplest example.
	document.getElementById( "elID" ).innerHTML = loadFile.sync('element.html');
//onload, onerror events.
	document.getElementById( "elID").innerHTML = loadFile.async( 'element.html',
	{
		onload: function ( response ) {

			var str = 'file has been loaded successfully';
			console.log( str );

		},
		onerror: function ( str, e ) {

			console.error( str );

		},
	}, );
