SpriteText.

A sprite based text component. SpriteText is a text that always faces towards the camera.

Quick start

  • Create a folder on your localhost named as [folderName].
    • Download three.js repository into your "[folderName]\three.js\dev" folder.
    • Download commonNodeJS repository into your "[folderName]\commonNodeJS\master" folder.
  • Add your web page into [folderName]. Example:

NOTE. Please include three.THREE = THREE; line into your project before use my library. See example above.

The easiest way to use SpriteText in your code is import SpriteText from SpriteText.js file in your JavaScript module.

import { SpriteText } from './commonNodeJS/master/SpriteText/SpriteText.js';

Now you can use SpriteText in your javascript code.

Add SpriteText with default settings into center of the scene.

scene.add( new SpriteText( 'Default sprite') );

Define the SpriteText parameters for all sprite texts you add to the scene. For example:

scene.userData.optionsSpriteText = {

	fontColor: 'rgba(0, 255, 0, 1)', //green
	textHeight: 0.2,
	fontFace: 'Times',

};

See options parameter of SpriteText for details.

Add new SpriteText.

const group = new THREE.Group();
scene.add( group );
group.add( new SpriteText( 'Sprite text', new THREE.Vector3( -4, 0, 0 ), {

	fontColor: 'rgba(255, 0, 0, 1)',//red

} ) );

SpriteTextGui

Add SpriteTextGui into dat.gui for manual change settings of the SpriteText. Example

Import dat.gui and add dat into three.

import { dat } from './commonNodeJS/master/dat/dat.module.js';
three.dat = dat;

Import SpriteTextGui.

import { SpriteTextGui } from './commonNodeJS/master/SpriteText/SpriteTextGui.js';

Now you can use SpriteTextGui in your javascript code.

const spriteTextGui = SpriteTextGui( scene );

Note. User can change text height, font face of all texts, but cannot change font color of second "Sprite text" because font color of second text is defined separatelly.

SpriteTextGui for group

User can change settings of the SpriteText for group separatelly. For example define another text height for second "Sprite text"

group.userData.optionsSpriteText = {

	textHeight: 0.1,

};

And add SpriteTextGui for group.

SpriteTextGui( group, spriteTextGui.userData.options, { spriteFolder: 'Sprite Text Setting for group' } );

folder.userData.restore() - Restore group.userData.optionsSpriteText to default values.

Example:

optionsSpriteText = {

	textHeight: 0.1,
	sizeAttenuation: false,

}
const fSpriteTextAll = SpriteTextGui( scene, options, {

	settings: { zoomMultiplier: 1.5, },
	options: optionsSpriteText

} );

//Change of the text height
optionsSpriteText.textHeight = 0.2;

//update of the options of all SpriteText, added in to group and all child groups
SpriteText.updateSpriteTextGroup( group );

//To do something...

//Restore options.textHeight to 0.1
fSpriteTextAll.userData.restore();
