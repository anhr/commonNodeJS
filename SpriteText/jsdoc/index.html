SpriteText.

A sprite based text component. SpriteText is a text that always faces towards the camera.

Quick start

    • Download three.js repository into your "[folderName]\three.js\dev" folder.
    • Download commonNodeJS repository into your "[folderName]\commonNodeJS\master" folder.

NOTE. Please include three.THREE = THREE; line into your project before use my library. See example above.

The easiest way to use SpriteText in your code is import SpriteText from SpriteText.js file in your JavaScript module.

import { SpriteText } from './commonNodeJS/master/SpriteText/SpriteText.js';

Now you can use SpriteText in your javascript code.

Add to scene a SpriteText with default settings.

scene.add( new SpriteText( 'Scene', new THREE.Vector3( 1, 0, 0 ) ) );

Please click and drag your mouse over canvas. You can see the "Scene" text that always faces towards the camera.

Define the SpriteText options for all sprite texts you add to the scene. For example:

scene.userData.optionsSpriteText = {

	fontColor: 'rgba(0, 255, 0, 1)', //green
	textHeight: 0.2,
	fontFace: 'Times',

};

See options parameter of SpriteText for details.

Plase call

SpriteText.updateSpriteTextGroup( scene );

after creating of all SpriteText instances for new scene.userData.optionsSpriteText have effect. Now you can see, your "Scene" text is green, text height is 0.2 and font name is "Times".

  • You can create a group of texts with specific settings.
const group = new THREE.Group();
group.userData.optionsSpriteText = {

	textHeight: 0.1,

};
scene.add( group );
const spriteTextGroup = new SpriteText( 'group', new THREE.Vector3( -4, 0, 0 ) );
group.add( spriteTextGroup );

All SpriteText instances, added into group, have text height is 0.1. Other settings takes from parent group i.e. from scene. You can see, child group setting have more priority before parent settings.

SpriteTextGui

Add SpriteTextGui into dat.gui for manual change settings of the SpriteText. Example

Import dat.gui and add dat into three.

import { dat } from './commonNodeJS/master/dat/dat.module.js';
three.dat = dat;

Import SpriteTextGui.

import { SpriteTextGui } from './commonNodeJS/master/SpriteText/SpriteTextGui.js';

Now you can use SpriteTextGui in your javascript code.

SpriteTextGui( scene, options );

Now you can see new "Sprite Text" folder in upper right corner of the canvas. User can change Height, Font Face and Font Color of the text.

Note. User can change Height of the "Scene" text, but not "Group" because group have another sprite text height. If you want the user can to change the height of the "Group" text, please add another SpriteTextGui.

SpriteTextGui( group, options, { spriteFolder: 'Sprite Text Setting for group' } );

You can add SpriteTextGui for changing options of each sprite test separately. For example edit the "Group" SpriteText for it.

const optionsIndividual = {

	fontColor: 'rgba(255, 0, 0, 1)',//red

}
const spriteTextGroup = new SpriteText( 'group', new THREE.Vector3( -4, 0, 0 ), optionsIndividual );

Now the "Group" text is red.

Add new SpriteTextGui.

SpriteTextGui( spriteTextGroup, options, {

	spriteFolder: 'Sprite with individual options',
	options: optionsIndividual,

} );

Now user can select another color of the "Group" text in the "Sprite with individual options" folder.
