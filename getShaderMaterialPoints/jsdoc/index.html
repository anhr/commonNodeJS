getShaderMaterialPoints.

I use getShaderMaterialPoints in my three.js projects for create THREE.Points with THREE.ShaderMaterial material.

The easiest way to use getShaderMaterialPoints in your code is import getShaderMaterialPoints from getShaderMaterialPoints.js file in your JavaScript module.

import getShaderMaterialPoints from './commonNodeJS/master/getShaderMaterialPoints/getShaderMaterialPoints.js';

Now you can use getShaderMaterialPoints in your javascript code. Example:

getShaderMaterialPoints( scene, [
	[],//first point. Zero position. White color.
	[0.5, 0.5, 0.5],//second point. White color.
], function ( points ) { scene.add( points ); } );
