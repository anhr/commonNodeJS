Class: FermatSpiral

FermatSpiral~FermatSpiral(settingsopt)

new FermatSpiral(settingsopt)

Implementation of Vogel's model of Fermat's spiral.
Parameters:
Name Type Attributes Default Description
settings object <optional>
 {} The following settings are available
Properties
Name Type Attributes Default Description
n number <optional>
 2 space dimension of Fermat's spiral.
count number <optional>
 500 points count.
c Float <optional>
 0.04 constant scaling factor. See Fermat's spiral for details.
position Array | number <optional>
 Array - position of Fermat's spiral in n-dimensional coordinates. 
number - position of the 0 coordinate of Fermat's spiral.
See settings.object.position parameter of the ND class.
