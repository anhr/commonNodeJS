Creates an array of graphic object faces.
Parameters:
Name Type Attributes Description
object THREE.Mesh You can see an array of faces in object.userData.intersections.faces after creating of Faces. See Mech.
collidableMeshList THREE.Mesh | array Mech or array of meshes, which intersects with object.
settings object <optional>
 the following settings are available:
Properties
Name Type Attributes Description
scene THREE.Scene <optional>
 THREE.Scene.
onReady function <optional>
 Callback function that called if intersection lines is ready and that take as input an array of all intersect lines. Every item of array is array of line vertices of THREE.Vector3 type.
