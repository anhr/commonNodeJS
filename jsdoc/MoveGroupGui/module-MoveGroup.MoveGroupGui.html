Class: MoveGroupGui

MoveGroup.MoveGroupGui(group, gui, optionsopt)

new MoveGroupGui(group, gui, optionsopt)

Add MoveGroup folder into dat.gui.
Parameters:
Name Type Attributes Description
group THREE.Group | THREE.Screen THREE group or scene for moving.
gui GUI is new dat.GUI(...).
options object <optional>
 the following options are available
Properties
Name Type Attributes Description
scales object <optional>
 axes scales. See AxesHelper( ... ) for details.
cookie Cookie <optional>
 Your custom cookie function for saving and loading of the MoveGroup settings. Default cookie is not saving settings.
cookieName string <optional>
 Name of the cookie is "MoveGroup" + options.cookieName. Default is undefined.
guiParams.getLanguageCode function <optional>
 Your custom getLanguageCode() function. returns the "primary language" subtag of the language version of the browser. Examples: "en" - English language, "ru" Russian. See the "Syntax" paragraph of RFC 4646 rfc4646 2.1 Syntax for details. Default returns the 'en' is English language. You can import { getLanguageCode } from '../../commonNodeJS/master/lang.js';
guiParams.lang object <optional>
 Object with localized language values
Properties
Name Type Attributes Description
moveGroup object <optional>
 use for rename of the MoveGroup folder. Default is 'Move Group'.
guiParams.guiSelectPoint GuiSelectPoint <optional>
 A dat.gui based graphical user interface for select a point from the mesh. See GuiSelectPoint for details.
