Properties
Name Type Attributes Description
tMin number <optional>
 start time. Uses for playing of the points. Default is 0.
name string <optional>
 Name of the points. Used for displaying of items of the Select drop down control of the Meshs folder of the dat.gui. Default is "".
shaderMaterial object <optional>
 creates the THREE.Points with THREE.ShaderMaterial material. The size of the each point of the THREE.Points seems the same on canvas because I reduce the size of the points closest to the camera and increase the size of the points farthest to the camera. See var shaderMaterialDefault of the frustumPoints for details.
position THREE.Vector3 <optional>
 position of the points. 
Vector's x, y, z is position of the points.
Can be as:
float - position of the points.
[float] - array of positions of the points.
Function - position of the points is function of the t. Example:
	new Function( 't', 'return 0.1 + t' )
Default is new THREE.Vector3( 0, 0, 0 ).
scale THREE.Vector3 <optional>
 scale of the points. 
Vector's x, y, z is scale of the points.
Can be as:
float - scale of the points.
[float] - array of scales of the points.
Function - scale of the points is function of the t. Example:
	new Function( 't', 'return 1.1 + t' )
Default is new THREE.Vector3( 1, 1, 1 ).
rotation THREE.Vector3 <optional>
 rotation of the points. 
Vector's x, y, z is rotation of the points.
Can be as:
float - rotation of the points.
[float] - array of rotations of the points.
Function - rotation of the points is function of the t. Example:
	new Function( 't', 'return Math.PI / 2 + t * Math.PI * 2' )
Default is new THREE.Vector3( 0, 0, 0 ).
arrayCloud array <optional>
 Array of points with cloud. 
If you define the array of points with cloud,
then you can define a points with cloud.
For example you can define
arrayCloud: options.arrayCloud
on the params of the
getShaderMaterialPoints( params, onReady )
function.
Or
arrayCloud: options.arrayCloud
on the pointsOptions of the myThreejs.points function.
Default is undefined
opacity boolean <optional>
 if true then opacity of the point is depend from distance to all meshes points from the group with defined mesh.userData.cloud. See options.getColors for details. Default is undefined.
onReady function <optional>
 Callback function that take as input the new THREE.Points.
